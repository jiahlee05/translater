# Conference Helper

A real-time speech recognition and translation web app for international conferences.

## Features
- Real-time speech recognition (Web Speech API)
- Instant translation (Google Translate)
- Camera overlay with subtitles
- Screenshot capture with gallery
- Translation history export (TXT)
- PWA Support (Installable)
- Dark Mode UI

## Technology Stack
- HTML5, CSS3, JavaScript (Vanilla)
- Web Speech API
- Google Translate API (Free endpoint)
- Canvas API

## Setup & Deployment
1. Clone repository.
2. Push to GitHub.
3. Enable GitHub Pages in Settings -> Pages -> Source: / (root).
4. Access via HTTPS (required for Camera/Mic).

## Usage
1. Allow Camera/Microphone permissions.
2. Select Source and Target languages.
3. Click "Start" to begin recognition.
4. Use "Capture" to take screenshots.
5. Use "Save TXT" in History to download logs.
